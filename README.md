# Fractured Flicks
A jigsaw puzzle web game where you put pieces of a video back together.

## Demo
https://mstop4.github.io/video-puzzle/

### Controls

* *Tap* - Rotate Piece  
* *Tap+Hold+Drag* - Move Piece 

Built with **Pixi.js**, **pixi-sound**, and **AWS S3**.
